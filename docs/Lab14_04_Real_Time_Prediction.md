# Lab 14.4 – Real-Time Prediction

## Objective

The objective of this laboratory is to perform real-time Brain–Computer Interface (BCI) prediction using the trained deep learning model developed in Chapter 12.

The extracted online EEG features are loaded into the trained Convolutional Neural Network (CNN), which predicts the user's intended command together with the confidence score for each prediction.

This laboratory represents the first complete real-time inference stage of the Hybrid Adaptive BCI system.

---

## Background

After signal acquisition, preprocessing, and feature extraction, the next stage of a real-time BCI system is classification.

A trained deep learning model receives the extracted EEG features and predicts the corresponding motor imagery class.

Unlike the offline experiments presented in Chapter 12, this laboratory demonstrates how the trained model can be used during real-time operation without retraining.

The prediction confidence provides an estimate of the model's certainty and is later used during adaptive decision making.

## Processing Pipeline

1. Load the extracted EEG feature matrix.
2. Load the trained CNN model.
3. Verify input compatibility.
4. Reshape feature vectors for Conv1D.
5. Perform real-time prediction.
6. Compute prediction confidence.
7. Save prediction results.
8. Generate prediction report.

## Discussion

The real-time prediction stage successfully classified every incoming EEG feature vector using the trained CNN model developed in Chapter 12.

Each prediction is associated with a confidence value generated by the Softmax output layer. These confidence values quantify the certainty of the classifier and provide an additional level of reliability before executing any control command.

Separating prediction from decision making improves system safety because the confidence values can later be evaluated before transmitting commands to external robotic systems.

This modular architecture allows the classifier to operate independently from the control layer, facilitating future integration with ROS2-based robotic platforms.

---

## Conclusion

The trained CNN model was successfully integrated into the real-time BCI pipeline.

The extracted EEG features were correctly reshaped, classified, and converted into prediction probabilities together with confidence scores.

The generated prediction results constitute the direct input for the next laboratory, where high-confidence predictions will be translated into executable control commands for robotic systems.

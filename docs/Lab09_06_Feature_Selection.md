# Lab 09.6 – Feature Selection

## Objective

The objective of this laboratory is to combine all extracted EEG feature groups into a single feature matrix and perform feature selection by removing low-variance features. This process reduces redundancy and prepares the dataset for machine learning.

---

## Background

Feature extraction often produces a large number of variables. However, not all features contribute equally to classification performance.

Feature selection improves model efficiency by removing features with little or no variation while preserving informative characteristics.

In this laboratory, a Variance Threshold method was applied to eliminate low-variance features before training machine learning models.

## Input Feature Groups

The following feature groups were combined:

- Time Domain Features
- Frequency Domain Features
- Power Spectral Density (PSD)
- Band Power Features
- Statistical Features

Each feature group was assigned a unique prefix:

| Prefix | Description |
|---------|-------------|
| TD_ | Time Domain |
| FD_ | Frequency Domain |
| PSD_ | Power Spectral Density |
| BP_ | Band Power |
| STAT_ | Statistical Features |

---

## Feature Selection Method

Method

```
VarianceThreshold
```

Threshold

```
1 × 10⁻¹²
```

This method removes features with extremely low variance while preserving informative variables.

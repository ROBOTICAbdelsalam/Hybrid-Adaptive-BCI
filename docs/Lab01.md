# Lab 01 – Environment Setup

## Overview

Lab 01 focused on preparing the software environment required for the development of the Hybrid-Adaptive-BCI project. A stable and reproducible Python environment is essential for EEG signal processing, machine learning, and future integration with ROS2.

This laboratory established the foundation for all subsequent stages of the project.

---

# Objectives

The main objectives of this laboratory were:

- Install Python and required scientific libraries.
- Configure the development environment.
- Install MNE-Python for EEG signal processing.
- Verify that all dependencies were working correctly.
- Prepare the project folder structure.

---

# Background

Developing Brain–Computer Interface (BCI) applications requires a reliable software environment capable of processing large EEG datasets and executing advanced machine learning algorithms.

Python has become the standard programming language for neuroscience research due to its rich ecosystem of scientific libraries.

The MNE-Python framework provides comprehensive tools for EEG preprocessing, visualization, filtering, artifact removal, and feature extraction, making it an ideal platform for this project.

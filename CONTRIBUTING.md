# Contributing

Thank you for your interest in the Hybrid-Adaptive-BCI project.

This repository is part of an academic Master's research project focused on developing a Hybrid Brain–Computer Interface (BCI) supported by Adaptive Artificial Intelligence for controlling industrial robots and medical assistive devices.

## How to Contribute

Contributions are welcome.

Please follow these guidelines:

- Follow Python coding standards (PEP 8).
- Keep the code clean and well documented.
- Write meaningful commit messages.
- Add comments where necessary.
- Save generated figures in the `figures/` folder.
- Save generated reports in the `results/` folder.
- Update the README when adding new features or laboratories.

## Reporting Issues

If you find a bug or have suggestions for improvements, please open an Issue describing the problem clearly.
